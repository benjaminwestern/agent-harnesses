# pi runtime bundle

This folder contains the pi-specific fixtures and prompts used by Agentic
Control.

Use the shared Go installer, `.artifacts/bin/agent_harness install --runtime pi`,
when you want a repo-local or global pi extension setup, and use the fixtures
and prompts here when you want to replay or exercise the shared contract.

Read the full runtime guide in [../../docs/pi.md](../../docs/pi.md).
