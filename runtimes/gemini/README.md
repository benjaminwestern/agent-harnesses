# Gemini runtime bundle

This folder contains the Gemini-specific fixtures and prompts used
by Agentic Control.

Use the shared Go installer, `.artifacts/bin/agent_harness install --runtime gemini`,
when you want a repo-local or global Gemini hook setup, and use the fixtures
and prompts here when you want to replay or exercise the shared contract.

Read the full runtime guide in [../../docs/gemini.md](../../docs/gemini.md).
