# OpenCode runtime bundle

This folder contains the OpenCode-specific fixtures, prompts, plugin source,
and plugin metadata used by Agentic Control.

Use the shared Go installer, `.artifacts/bin/agent_harness install --runtime opencode`,
when you want a global or repo-local OpenCode plugin setup, and use the
fixtures and prompts here when you want to replay or exercise the shared
contract.

Read the full runtime guide in [../../docs/opencode.md](../../docs/opencode.md).
