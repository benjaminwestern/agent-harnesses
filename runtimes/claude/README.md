# Claude runtime bundle

This folder contains the Claude-specific fixtures and prompts used
by Agentic Control.

Use the shared Go installer, `.artifacts/bin/agent_harness install --runtime claude`,
when you want a repo-local or global Claude hook bundle, and use the fixtures
and prompts here when you want to replay or exercise the shared contract.

Read the full runtime guide in [../../docs/claude.md](../../docs/claude.md).
